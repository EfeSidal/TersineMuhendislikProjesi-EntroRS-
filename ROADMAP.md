# Roadmap | Yol Haritası

## Milestones | Kilometre Taşları (English)

### v1.0.0 — Completed | Tamamlandı ✅
- **PE Parsing**: Full support for Windows Portable Executable (Base header).
- **Shannon Entropy**: Mathematical engine for calculating data randomness.
- **IAT Analysis**: Identification of external dependencies (Import Address Table).
- **Strings**: Basic identification of static indicators.

### v1.1.0 — Planned | Planlanan 🚀
- **Enhanced Reporting**: Add support for **JSON** and **HTML** format exports for better integration.
- **Improved IAT**: Detailed library explanations and more suspicious API mappings.

### v1.2.0 — Planned | Planlanan 🚀
- **YARA Rule Integration**: Support for scanning binaries using custom YARA rules.
- **Signature Detection**: Identifying known packers like UPX or VMProtect by signature.

### v2.0.0 — Long-term | Uzun Vade 📅
- **Full ELF (Linux) Support**: Extensive parsing and analysis of ELF (Executable and Linkable Format) binaries.
- **Universal Static Engine**: Support for multiple file formats with a unified heuristic engine.

## Kilometre Taşları (Türkçe)

### v1.0.0 — Completed | Tamamlandı ✅
- **PE Analizi**: Windows Portable Executable (PE) başlık desteği.
- **Shannon Entropisi**: Veri rastgeleliği hesaplama motoru.
- **IAT Analizi**: Dış bağımlılıkların (Import Address Table) tespiti.
- **Strings**: Statik dizgilerin temel tespiti.

### v1.1.0 — Planlanan 🚀
- **Gelişmiş Raporlama**: Daha iyi entegrasyon için **JSON** ve **HTML** rapor çıktısı desteği.
- **Geliştirilmiş IAT**: DLL açıklamaları ve daha fazla şüpheli API eşleşmesi.

### v1.2.0 — Planlanan 🚀
- **YARA Kuralı Entegrasyonu**: Özel YARA kuralları ile binary tarama desteği.
- **İmza Tespiti**: UPX, VMProtect gibi paketleyicilerin imza ile tanımlanması.

### v2.0.0 — Uzun Vade 📅
- **Tam ELF (Linux) Desteği**: ELF (Executable and Linkable Format) dosyaları için detaylı analiz motoru.
- **Evrensel Statik Motor**: Tek bir motor ile birden fazla dosya formatına destek.
